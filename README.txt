General Usage Notes
---------------------------------------------------
-This program is used for the Udacity Part 1 Project. The program consists of three python files: entertainment.py, media.py, and fresh_tomatoes.py. Each is integral for the program to function. media.py creates and initializes a class Movie. entertainment.py defines each instance of class Movie. fresh_tomatoes is used to create the web-facing aspect of the website and each instance.
